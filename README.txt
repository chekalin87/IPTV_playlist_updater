- обновление исходников
- формирование исходных плейлистов
- формирование плейлиста мечты
- просмотр ссылок в твоём плеере

    Как работать?:
В links.txt нужно положить прямые ссылки на плейлисты (каждую с новой строки). --ВРЕМЕННО
Запускаешь прогу, выбираешь источник(по умолчанию выбран объединённый источник (all), состоящий из всех плейлистов),
выбираешь канал, справа будет ссылка(можно выбрать если их будет несколько), можно добавлять, удалять, перемещать, ну и
смотреть. В меню "Файл" можно открыть и экспортировать лист.
"UPDATE" перезагрузит файлы и обновит исходные листы.
